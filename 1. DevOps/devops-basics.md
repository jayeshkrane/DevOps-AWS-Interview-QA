# 🔧 DevOps Basics

## ❓ What is DevOps?
DevOps is a combination of Development and Operations aimed at improving collaboration, automation, and faster delivery of software.

## 🔄 What is CI/CD?

### CI (Continuous Integration)
- Developers push code frequently
- Code is automatically built & tested

### CD (Continuous Delivery/Deployment)
- Code is automatically deployed to production

## 📊 SDLC Phases
1. Planning  
2. Development  
3. Testing  
4. Deployment  
5. Monitoring  

## ⚡ Benefits of DevOps
- Faster delivery  
- Automation  
- Reduced errors  
- Better collaboration  

## 🔥 Interview Questions

**Q1: What is DevOps?**  
👉 DevOps is a culture that automates and integrates development and operations.

**Q2: Difference between CI and CD?**  
👉 CI = Build & Test  
👉 CD = Deploy
