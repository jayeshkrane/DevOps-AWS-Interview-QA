# ☁️ AWS Basics

## 🔹 What is AWS?
Amazon Web Services is a cloud platform providing infrastructure services.

## 🔑 Core Services

### EC2
- Virtual server

### S3
- Storage service

### IAM
- User & permission management

### VPC
- Private network in cloud

### Load Balancer
- Distributes traffic

### Auto Scaling
- Automatically adjusts servers

## 🔥 Interview Questions

**Q1: What is EC2?**  
👉 Virtual machine in AWS.

**Q2: What is S3?**  
👉 Object storage service.

**Q3: What is IAM?**  
👉 Manages users and permissions.
